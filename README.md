## Testimony Feed

Esse é um projeto front end feito com React e Vite. Um feed de depoimentos e recomendações, ainda estático, (o comentário irá funcionar, mas, ao atualizar a tela eles sumirão).

>Para instalar:

```sh
npm install

```

>Para rodar:

```sh
npm run dev

```
